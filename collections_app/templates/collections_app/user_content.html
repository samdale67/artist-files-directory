About Directory Records

  • The directory currently contains {{ collector_count }} collector records, {{ collection_count }} collection records, {{ image_count }} images, and {{ document_count }} documents.
  • Directory entries are pairings of collector records and collection records. A collector can "own" multiple collections. Similarly, a collection can have multiple collectors, which is the case for digital collections that contain material from multiple collectors. See Missouri Remembers: Artists in Missouri 1821-1951.
  • With that in mind, you'll first need to create a collector record before you can associate a collection. After creating a collector record, you will return to this page and create a collection record and connect a collector (or collectors).
  • Any authenticated user can add and update the following fields: institution type, private collector type, reference service, cataloging system, special format, subject: name, subject: topic, subject: city, subject: county, subject: states and provinces, subject: country, subject, and geographic area.
